Developing Games in Unity, Part 1

A "Hello World" Equivalent

Create a new Unity Project named "Hello World" or something similar. Do not import any packages, and setup defaults for your chosen path - 2D or 3D.

For the 3D Path:

Create a scene with a directional colored light and a simple cube with a texture that rotates along one or more axis using a script component written in C#. Experiment with multiple speeds of rotation.

If you have additional time, add mouse or keyboard functionality to change the axis of rotation during run-time (using the same script) or to move the cube in eight directions (using a separate script).

For the 2D Path:

Create a scene with a simple sprite with a texture that changes upon mouse click or button press using a script component written in C#.

If you have additional time, implement keyboard functionality that moves the sprite using mouse or keyboard input using a separate script.
