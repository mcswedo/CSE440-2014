Tutorials In Unity

Tutorial Project

Reminder that the Tutorials are due Thursday evening. Please upload the entire project folder (including the Assets, Library, Project Settings folders and all files) to GitHub before the deadline for full credit.

Following are links to several tutorials spanning several genres of video games, all of which teach both basics of Unity development and also get into more advanced topics. Please choose the tutorial that is most similar to the game project you wish to work on, as it will teach you some required tools, techniques, and programming skills applicable to your project.

Stealth - a 3D isometric, third-person stealth action game

Stealth is one of two tutorial projects created and maintained by the Unity developers themselves, and demonstrates how an isometric game functions. Properties of this game can be applied to both third- and first-person 3D games as well, and the logic demonstrated is incredibly useful for 3D development.

Space Shooter - an orthographic, arcade-like space shooter using 3D assets

Space Shooter is the other tutorial project currently available from the Unity developers, and it details how an orthographic (zero perspective), top-down space shooter can be easily created in the Unity engine. Fundamentals taught in this tutorial can be used in both 3D and 2D development of side-scrollers, platformers, and obviously arcade-style shooters.

Zombie Conga - a 2D, top-down, mouse (or touch) input based tutorial on Unity's 2D toolkit

Zombie Conga's tutorial is a very detailed look into Unity's 2D functionality, and explains many things about how Sprites are handled, how to use 2D objects in Unity's environment, and its lessons can also be applied to mobile game development. It is an incredibly powerful tutorial for 2D development, and can be applied to side scrollers, platformers, and any and all 2D, sprite-based game concepts.

This "Getting Started" tutorial linked above is merely the first of five tutorials in the Zombie Conga Tutorial. The follow up lessions (which are also required, if you choose this option) can be found through these following links: Part 2 - Animations, Part 3 - Animation Controllers, Part 4, Physics and Screen Sizes, and Part 5, Scrolling scenes and sounds.

Runner - a 2.5D 'infinite' platformer

Runner is a tutorial that requires prior knowledge of how Unity works, and does much less step-by-step work and more broad-stroke work. As such, it is more challenging for a beginner developer. However, because Runner covers such concepts as procedural generation and physics-based platformer movement controls as well as using such tools as delegates (which are incredibly useful but more advanced topics), I am including it. The game can be likened to games such as Canabalt or Robot Unicorn Attack - an "infinite runner" game. It primarily explains 2.5D platformer development (IE, flat-plane movement with perspective) and can be applied to mobile-based games as well.

Additionally, I feel that his folder structure is rather disorganized, and would highly recommend you to use a standard "Scripts," "Scenes," "Materials," "Prefabs" folder style, like in the other tutorials. I also disagree with several conventions he puts forth in this tutorial, such as assigning a public script and referencing it via the Inspector - it is much less safe than creating instances of scripts using tags / the "GetComponent" function.
